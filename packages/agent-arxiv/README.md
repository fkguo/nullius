# @nullius/agent-arxiv

Reserved namespace for a future arXiv-oriented agent/community surface. **No runtime implementation yet** — the package currently exports only `VERSION` so the workspace name is held while the design is planned.

Roadmap and design are tracked in maintainer-local notes under `~/.nullius-dev/`; this package is intentionally absent from any public surface advertised in the root `README.md`. Until a real entrypoint lands here, do not import from `@nullius/agent-arxiv` outside the workspace and do not treat it as a stable contract.

See the root [AGENTS.md](../../AGENTS.md) §Stable Public Invariants for the "no half-finished implementations as public authority" rule that scopes this placeholder.
